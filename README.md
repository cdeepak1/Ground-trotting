# Ground Truthing and Spectroscopy Assignment

## Overview

This repository contains the results and analysis conducted as part of the course "Sampling Design & Ground Truthing in Remote Sensing" (Module: 12-GEO-M-RS02) at the Faculty of Physics and Earth System Sciences, SoSe 24.
The assignment focuses on species identification, spectral measurements of flowers and leaves, and subsequent analysis to determine which spectral dataset (flowers or leaves) is more suitable for species discrimination.
